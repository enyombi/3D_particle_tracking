In the directory are all the datafiles and programs for generating ../movies_particleTracking/movie30may2014_1_30mayRCP_4xALL.mp4

See notesPovrayAnimation.txt for more details

movie_p.ini uses povray to create a numerous *.pov and then a movie

povray itself can be used to create a a single-frame of the movie, which is a single image of the packing like ../packing30may2014_1_30mayRCP_4x.png.  You'll need to install povray, generate a .pov file, and then run povray w/that .pov file. For example:

$ povray ().pov
